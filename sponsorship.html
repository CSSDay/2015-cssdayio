About CSS Day

CSS Day is an all-day event on Saturday, December 5th 2015 that will showcase advanced CSS lectures for experienced web developers. CSS Day's highly skilled audience can expect a fun and friendly atmosphere with exciting talks and networking opportunities.

Front-end web development is a fascinating topic with a surge of interest in today's job market. Web developers today are required to possess a more complex set of skills than ever before to be successful. The conference will cater to the engineering mindset while bringing clarity to the complexities of CSS methodologies, best practices, and lesser known tricks.

Event Features

CSS Day will attract web developers with an eventful day featuring:

  • Lectures and Workshops
  • Free Lunch (see sponsorship opportunity)
  • Swag and T-Shirt Giveaways
  • A CSS coding contest (see sponsorship opportunity)
  • An After Hours Party! (see sponsorship opportunity)

Highly Technical Audience

CSS Day is a unique opportunity to meet advanced front-end and back-end developers alike who will be in attendance. The audience makeup will be mid-level to senior JavaScript, Ruby, Python, Java, and .NET developers.

For our inaugural event, CSS Day is planning on having between 100 and 200 of Phoenix's top developers in attendance. Tickets are currently being sold and 70 early-bird tickets were sold out within one week.
